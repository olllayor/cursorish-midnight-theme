# Changelog

All notable changes to the Cursorish Midnight theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.3] - 2025-11-01

### Changed (0.0.3)

- Refined default foreground and editor text colors for stronger contrast on midnight backgrounds
- Smoothed hover and selection states across lists, tabs, menus, and the activity bar to prevent harsh color jumps
- Brightened global icon colors and added explicit icon treatments for hover and selection states so glyphs stay visible
- Increased status bar, title bar, and sidebar text contrast for better legibility in low-light setups

### Fixed (0.0.3)

- Resolved cases where icons disappeared or became hard to see when hovered or activated throughout the workbench

## [0.0.2] - 2025-10-30

### Added (0.0.2)

- Extension icon (icon.jpg) for marketplace display
- Comprehensive semantic token colors for enhanced syntax highlighting
- Full support for modern VS Code features including inlay hints
- Bracket highlighting with color-coded pairs
- Optimized diff editor colors

### Fixed (0.0.2)

- Improved package configuration for marketplace visibility
- Enhanced color contrast for better readability in midnight environments

## [0.0.1] - 2025-10-30

### Added (0.0.1)

- Initial release of Cursorish Midnight theme
- Comprehensive ultra-dark color palette optimized for night-time coding
- Full syntax highlighting support for all major programming languages:
  - JavaScript / TypeScript
  - Python
  - Java
  - C / C++
  - HTML / CSS
  - Go
  - Rust
  - JSON / YAML
  - Markdown
  - And many more
- Semantic token colors for enhanced code highlighting
- Complete UI colors for all VS Code interface elements:
  - Activity bar
  - Editor UI
  - Sidebar
  - Status bar
  - Panels and terminals
- Terminal color palette (ANSI colors)
- Bracket highlighting colors (6-level rainbow)
- Diff highlighting colors (added, removed, modified)
- Git decoration colors
- Error and warning indicators
- README with installation and customization instructions
- MIT License

### Features (0.0.1)

- Ultra-dark background for minimal eye strain during night coding sessions
- Carefully selected accent colors optimized for dark environments
- Vibrant syntax highlighting for excellent code visibility
- Consistent styling across all UI elements
- Optimized for extended coding sessions
- Support for all major languages and frameworks
